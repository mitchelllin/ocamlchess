Module Main

module Main: sig .. end

The main entry point for the chess interface.

val prompt_msg : string

prompt_msg is a string that prompts an action.

val invalid_ins_msg : string

invalid_ins_msg is a string that denotes an invalid command while within the instructions portion of the game.

val invalid_ins_next : string

invalid_ins_next is a string that denotes the end of instructions.

val invalid_cmd_msg : string

invalid_msg is a string that denotes an invalid command.

val invalid_start_msg : string

invalid_start_msg is a string suggesting to type 'Start <timer>'.

val invalid_timer_msg : string

invalid_timer_msg is a string suggesting a number > 0'.

val invalid_view_msg : string

invalid_view_msg is a string suggesting to type 'View instructions'.

val execute_promotion : State.t -> Piece.rank -> (string -> State.t -> unit) -> unit

execute_promotion checks if st has a pawn that can be promoted to rank. If so, executes play_msg on a new st' representing st with the pawn promoted. Otherwise, repeats the turn of st.

val execute_make : string list -> State.t -> (string -> State.t -> unit) -> unit

execute_make checks if lst contains a valid promotion rank, and if so, calls execute_promotion to attempt the promotion.

val check_status : State.t -> State.t -> (string -> State.t -> unit) -> unit

check_status checks the check/checkmate status of st and st', and executes play_msg accordingly, notifying the player if there exists some kind of check/checkmate status.

val dec_time : float -> float -> Piece.team -> float -> float -> float * float

dec_time returns (w_time', b_time') which depict the time each team has left to play.

val read_input : string -> State.t -> (string -> State.t -> unit) -> unit

read_input parses user input i on state st and prints the resulting state st' with any changes, should a change occur.

val play_turn_helper : State.t -> (string -> State.t -> unit) -> unit

play_turn_helper exits the game if the current turn's team of st in checkmate. Otherwise, prompts for user input.

val play_turn : Piece.team -> State.t -> float -> float -> float -> unit

play_turn prints the entirety of the game gui, depicting both st and the teams' remaining times, w_time and b_time.

val execute_instructions : int -> (string -> unit -> unit) -> string -> unit -> unit

execute_instructions prints the instructions page at index and prompts for user input.

val start_reply : (string -> unit -> unit) -> string -> string list -> unit

start_reply starts the chess game if the argument passed by start is "game". Otherwise, reprints the home screen.

val view_reply : (string -> unit -> unit) -> string -> string list -> unit

view_reply prints the first instructions page if the argument passed by view is "instructions". Otherwise, reprints the home screen.

val execute_start : string -> unit -> unit

execute_start prints the onboarding GUI and prompts for user input.

val main : unit -> unit

main starts the chess simulation, calling execute_start.

main () starts the chess game, by prompting its home/onboarding page.

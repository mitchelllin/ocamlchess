Module Board

module Board: sig .. end

Representation of static chess game board.

type t = Piece.piece list

t represents the entire chess board, with every chess piece's current properties.

t represents the board, ie. the pieces on the board, and their respective properties.

val strings_to_pieces : int * int -> t -> string list -> t

strings_to_pieces returns a board with pieces represented in the string list lst.

val file_to_board : string list -> string -> t

file_to_board converts the contents of d1/.../dn/file_name to a board, where d1 ... dn are folder names denoted in dir_lst.

file_to_board d_lst f returns a board represented by the text of file path d1/.../dn/f, in which d_lst holds d1...dn.

val init_board : t

init_board returns a list of chess pieces in their respective initial states.

init_board initializes the board, returning a list of all chess pieces for both teams in their starting positions.

val get_piece : (char * int) option -> Piece.piece list -> Piece.piece option

get_piece returns a piece p in bd such that get_position p = pos.

get_piece pos bd returns the piece that occupies position pos on the board bd.

val cmp_boards : 'a list -> 'a list -> bool

cmp_boards returns true if bd1 is the same as bd2, where order of pieces do not matter. Otherwise, returns false.

cmp_boards bd1 bd2 returns true if the set like lists bd1 and bd2 are equivalent. Otherwise, returns false.

val add_piece : 'a -> 'a list -> 'a list

add_piece returns bd with p added.

add_piece p bd returns a new board with the piece p added.

val remove_piece : (char * int) option -> Piece.piece list -> t

remove_piece returns bd without the piece at pos.

remove_piece pos bd returns a new board with the piece at pos removed.

val update_board : Piece.piece -> (char * int) option -> Piece.piece list -> t

update_board p pos bd returns a new board with p moved to pos.

update_board p pos bd returns a board bd' with p moved to pos.

val is_piece_at : (char * int) option -> Piece.piece list -> bool

is_piece_at returns true if there is a piece at pos. Otherwise, returns false.

is_piece_at pos bd returns true if there is a piece at pos within bd. Otherwise, returns false.

val is_clear : 'a list -> 'b * 'c -> 'd * 'e -> ('b * 'c -> 'd * 'e -> 'a -> bool) -> bool

is_clear lst (col1, row1) (col2, row2) is_in_range returns whether a list of pieces lst contains one or more pieces in between the line drawn from (col1,row1) to (col2,row2) using the is_in_range function. This method is a helper for checking if horizontal, vertical, and diagonal paths are clear.

val is_in_horizontal : char * int -> char * int -> Piece.piece -> bool

is_in_horizontal (col1, row1) (col2, row2) p returns if a piece is located in the horizontal path between (col1,row1) and (col2,row2).

val is_in_vertical : char * int -> char * int -> Piece.piece -> bool

is_in_vertical (col1, row1) (col2, row2) p returns if a piece is located in the vertical path between (col1,row1) and (col2,row2).

val is_in_diagonal : char * int -> char * int -> Piece.piece -> bool

is_in_diagonal (col1, row1) (col2, row2) p returns if a piece is located in the diagonal path between (col1,row1) and (col2,row2).

val is_clear_vertical : t -> Piece.position -> Piece.position -> bool

is_clear_vertical bd pos1 pos2 returns if the vertical path spanned by pos1 and pos2 is clear of pieces on bd.

is_clear_vertical bd pos1 pos2 is true if the vertical path spanned by pos1 and pos2 is clear of pieces on bd. Otherwise, is false.

val is_clear_horizontal : t -> Piece.position -> Piece.position -> bool

is_clear_horizontal bd pos1 pos2 returns if the horizontal path spanned by pos1 and pos2 is clear of pieces on bd.

is_clear_horizontal bd pos1 pos2 is true if the horizontal path spanned by pos1 and pos2 is clear of pieces on bd. Otherwise, is false.

val is_clear_diagonal : t -> Piece.position -> Piece.position -> bool

is_clear_diagonal bd pos1 pos2 returns if the diagonal path spanned by pos1 and pos2 is clear of pieces on bd.

is_clear_diagonal bd pos1 pos2 is true if the diagonal path spanned by pos1 and pos2 is clear of pieces on bd. Otherwise, is false.

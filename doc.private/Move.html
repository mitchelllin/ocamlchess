Module Move

module Move: sig .. end

Representation of movement validity checks.

val is_vertical_move : Piece.position -> Piece.position -> bool

is_vertical_move is true if pos1 and pos2 hold the same column and have different rows. Note: Does not check if pos1 and pos2 are legal positions on a board.

is_vertical_move pos1 pos2 is true if pos1 to pos2 represents a vertical move. Otherwise, is false.

val is_horizontal_move : Piece.position -> Piece.position -> bool

is_horizontal_move is true if pos1 and pos2 hold the same row and have different columns. Note: Does not check if pos1 and pos2 are legal positions on a board.

is_horizontal_move pos1 pos2 is true if pos1 to pos2 represents a horizontal move. Otherwise, is false.

val is_diagonal_move : Piece.position -> Piece.position -> bool

is_diagonal_move returns true if pos1 and pos2 have equal differences between its rows and columns. Note: Does not check if pos1 and pos2 are legal positions on a board.

is_diagonal_move pos1 pos2 is true if pos1 to pos2 represents a diagonal move. Otherwise, is false.

val pawn_valid_move : Piece.piece -> Piece.position -> Piece.position -> bool

pawn_valid_move returns true if a pawn p can move from pos1 to pos2. Otherwise, returns false. Note: pawn_valid_move is only for basic pawn movement, and does not account for diagonal capturing of an opposing piece (this is done in state.ml).

val king_valid_move : Piece.position -> Piece.position -> bool

king_valid_move returns true if pos1 moving to pos2 represents a one-space move diagonally, vertically, or horizontally. Otherwise, returns false.

val knight_valid_move : Piece.position -> Piece.position -> bool

knight_valid_move returns true if pos1 moving to pos2 represents a L-shaped move of 2-to-1 ratio either horizontally then vertically, or vertically then horizontally. Otherwise, returns false.

val queen_valid_move : Piece.position -> Piece.position -> bool

queen_valid_move returns true if pos1 moving to pos2 is a horizontal, vertical, or diagonal move. Otherwise, returns false.

val rook_valid_move : Piece.position -> Piece.position -> bool

rook_valid_move returns true if pos1 moving to pos2 is a horizontal or vertical move. Otherwise, returns false.

val is_valid_basic_move : Piece.piece -> Piece.position -> bool

is_valid_basic_move returns true if p can move to pos2, where pos2 is not its current position. Otherwise, returns false.

is_valid_basic_move p pos2 returns true if the piece p can move to pos2 as defined by chess rules. Otherwise, returns false.

val valid_castle : Piece.piece -> Piece.piece -> Board.t -> bool

valid_castle returns true if:

  • there exists p1 and p2 that have not moved yet.
  • one of p1 and p2 is a rook, and the other is a king.
  • there are no pieces in between p1 and p2 horizontally on bd. Otherwise, returns false.

valid_castle p1 p2 bd returns true if the piece p1 can castle with p2 under castling rules. Otherwise, returns false.

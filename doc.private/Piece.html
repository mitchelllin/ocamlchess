Module Piece

module Piece: sig .. end

Representation of a chess piece.

type rank =
| Pawn
| King
| Queen
| Knight
| Rook
| Bishop

The type of chess piece ranks.

type team =
| White
| Black

The type of teams within a game.

type position = (char * int) option

The type of location identifiers.

type piece = {
   position : position;
   rank : rank;
   team : team;
   is_first_move : bool;
}

The type of chess pieces.

val get_position : piece -> position

get_position returns the position of p.

get_position p returns the position of a given piece p denoted by Some (row, col) if the piece has not been captured, otherwise None.

val get_rank : piece -> rank

get_rank returns the rank of p.

get_rank p returns the rank of a given piece p.

val get_team : piece -> team

get_team returns the team of p.

get_team p returns the team of a given piece p.

val is_first_move : piece -> bool

is_first_move returns true if p has not been moved, else is false.

is_first_move p returns true if a player has not moved their piece p yet. Otherwise, returns false.

val set_piece : position -> rank -> team -> bool -> piece

set_piece returns a piece with position pos, rank r, team team, is_first_move fm.

set_piece pos r team fm returns a piece with position pos, rank r, team team, and is_first_move fm.

val set_position : piece -> position -> piece

set_position returns a piece p' with the same fields as p, but pos for position, and false for is_first_move.

set_position p pos returns a new piece similar to p, with the given position pos.

val positions_equal : ('a * 'b) option -> ('a * 'b) option -> bool

positions_equal returns true if pos1 and pos2 are structurally equal, and both are not None.

val is_valid_position : position -> bool

is_valid_position returns true if pos is a possible position on the board, ie. it is Some (col, row) such that col is a character between a and h, and row is an integer between 1 and 8 inclusively. Otherwise, returns false.

is_valid_position returns true if pos is a valid position on a chess board. Otherwise, returns false.

val is_pawn : piece -> bool

is_pawn returns true if p is a Pawn. Otherwise, returns false.

is_pawn p returns true if p is a pawn. Otherwise, returns false.

val is_blockable : piece -> bool

is_blockable returns true if p is a piece with a blockable path which includes Rook, Bishop, and Queen. Otherwise, returns false.

is_blockable returns true if p is a piece with a blockable path which includes Rook, Bishop, and Queen. Otherwise, returns false.

val string_to_rank : string -> rank option

string_to_rank returns the rank of str.

string_to_rank str returns the rank corresponding to str as Some rank. If str does not convert to a rank, then is None.

val rank_to_string : rank -> string

rank_to_string returns the string of rank.

rank_to_string r returns the rank in string format of a given rank r.

val team_to_string : team -> string

team_to_string returns the string of team.

team_to_string t returns the team in string format of a given team t.

val position_to_string : (char * int) option -> string

position_to_string returns pos as a string if pos is Some (col, row). Returns "No position" if pos is None.

position_to_string pos returns the string format "(col, row)" of pos if pos is Some (col, row). Otherwise, returns "No position".

val piece_to_string : rank -> team -> string

piece_to_string returns the string form of a chess piece, determined by r and t.

piece_to_string r t returns a piece of rank r and team r as a string.

val string_to_piece : position -> string -> piece option

string_to_piece returns Some p where p is a piece with is_first_move as true, rank and team of str, and position pos, if str denotes a chess piece. Otherwise, returns None.

string_to_piece pos str returns a string str as a piece at position pos.

val get_surrounding_positions : (char * int) option -> position list

get_surrounding_positions returns a list of valid board positions that are one unit away from pos veritically, horizontally, and diagonally.

get_surrounding_positions pos returns all positions on a board that surround the position pos.

val order_positions : position -> position -> position * position

order_positions returns (pos, pos') where pos is whichever position of pos1 or pos2 comes first, and pos' being the latter.

val inc_position : (char * int) option -> int * int -> (char * int) option

inc_position returns a position pos' which is pos with its column and row offset by (c_offset, r_offset).

val build : (char * int) option ->
       (char * int) option ->
       int * int -> (char * int) option list -> (char * int) option list

build returns a list of positions strictly between from_pos, and to_pos, using inc to determine the incrementation of positions.

val get_positions_between : position -> position -> position list

get_positions_between returns a list of positions strictly between pos1 and pos2. Returns empty list if pos1 and pos2 are not horizontal, vertical, or diagonal positions.

get_positions_between pos1 pos2 returns all positions on a board that are between the position pos1 and position pos2 along a straight line.
